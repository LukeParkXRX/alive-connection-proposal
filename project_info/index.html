Project Overview

프로젝트 개요

NFC/BLE 기반 자동 명함 교환과 AI 관계 인사이트를 결합한 차세대 네트워킹 플랫폼

ALIVE Connection은 NFC 탭 한 번으로 양방향 디지털 명함을 교환하고, 만남의 컨텍스트(장소, 시간, 메모)를 자동 기록하며, AI 기반 관계 인사이트를 제공하는 크로스 플랫폼 네트워킹 앱입니다.

Architecture

기술 아키텍처

BLE Primary + Server Exchange + NFC Boost 3-Layer Hybrid Architecture

📡 Layer 1
BLE Discovery (Primary)
Bluetooth Low Energy — 항상 활성
BLE 스캔 + 광고를 통한 기기 발견, 앱 보유 확인, userId 교환. RSSI 근접 필터(-50dBm)로 근거리 사용자만 감지. 5m 이내 자동 트리거.
☁️ Layer 2
Server Exchange
Express.js + Supabase — 데이터 교환 핵심
프로필 데이터 교환, 만남 기록 저장, GPS 위치 태깅, 역지오코딩. JWT 인증 기반. 양측 동시 교환 보장.
📳 Layer 3
NFC/HCE Boost (Optional)
Near Field Communication — 보조 채널
Android HCE 즉시 교환, iOS CoreNFC 태그 읽기. 앱 미설치 상대 대응(딥링크/NDEF). BLE 실패 시 폴백 채널.
⚙️
ExchangeManager — 통합 조율기
BLE, NFC, Server 3개 레이어를 통합 관리하는 싱글톤 클래스. 상태 머신 기반(idle → scanning → discovered → exchanging → completed)으로 교환 플로우를 조율하며, 실패 시 자동 폴백과 오프라인 큐를 처리합니다.
